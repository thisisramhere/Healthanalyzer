# 🧠 Healthalyzer – AI Stroke Risk Prediction System

Healthalyzer is an AI-powered application designed to predict the probability of stroke occurrence using deep learning techniques. By analyzing key health parameters, the system provides users with an early risk assessment, enabling proactive healthcare decisions.

---

## 🚀 Features

- 🔍 Predicts stroke risk using a trained CNN model  
- 📊 Accepts user inputs such as:
  - Age  
  - Blood Pressure  
  - Glucose Level  
  - BMI  
  - Lifestyle habits (smoking, activity, etc.)  
- ⚡ Real-time prediction output  
- 🧠 Deep learning-based analysis for improved accuracy  

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries & Frameworks:**  
  - TensorFlow / Keras  
  - NumPy  
  - Pandas  
  - Scikit-learn  
- **Frontend (if applicable):** HTML, CSS, JavaScript  
- **Environment:** Google Colab / Jupyter Notebook  

---

## 🧠 Model Details

- Model Type: Convolutional Neural Network (CNN)  
- Input Features:
  - Age  
  - Blood Pressure  
  - Glucose Level  
  - BMI  
  - Lifestyle factors  
- Output:
  - Probability score indicating stroke risk
